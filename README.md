## Installation

```bash
composer require attla/laravel-html-minify
```

## Usage

```env

# For enable 
HTML_MINIFY = true

# For disable
HTML_MINIFY = false

# For preserve HTML comments
HTML_MINIFY_COMMENTS = true

# For preserve HTML conditional comments
HTML_MINIFY_CONDITIONAL_COMMENTS = true

```
